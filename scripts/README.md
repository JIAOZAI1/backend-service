# scripts

仓库统一运维/构建/开发脚本目录，禁止在此目录之外散落脚本。

| 子目录 | 用途 |
| --- | --- |
| `dev/` | 本地开发、调试、联调 |
| `build/` | 构建服务、构建镜像、打包产物 |
| `deploy/` | 部署、回滚、发布 |
| `db/` | 数据库迁移、初始化、回滚 |
| `ops/` | 运维操作（备份、清理日志、健康检查等） |
| `ci/` | CI/CD 流水线辅助 |

命名规范：`动词-对象.sh`，如 `build-image.sh`、`deploy-service.sh`、`backup-db.sh`。

详见根目录 [微服务项目开发规范.md](../微服务项目开发规范.md) 第 11-12 章。
